# Debian-Packages

A few Debian packages for configuration, scripts and not yet packaged sources will be pushed here.

## First package

- **anduinos-butterfly-config** : Post-installation configuration package for AnduinOS Butterfly Fr,
  a French-language remix of AnduinOS (Ubuntu based) with additional tweaks.
